# Milestone 2: KPI Engineering and Dashboard Preparation

## Overview

In this milestone, the cleaned dataset was enriched by generating key performance indicators (KPIs) and adding metadata such as region and alliance information. The dataset was then prepared in a format suitable for visualization and the structure of the dashboards was planned.

---

# Objective

The main goals of this milestone are:

* Generate analytical KPIs from the cleaned military dataset
* Enrich the dataset with geographic and alliance information
* Prepare the dataset for visualization tools such as Power BI
* Design and prototype the structure of the dashboards

---


## Work Done

* Generated analytical KPIs including **Power Index Rank Gap**, **Assets per Capita**, and **Budget-to-GDP Ratio** from the cleaned dataset.
* Processed the dataset using Python to compute these indicators for all countries.
* Added additional metadata such as **region**, **continent**, and **alliance indicators** to improve filtering and comparisons.
* Structured the dataset to support **country-level comparisons and regional analysis**.
* Organized the data in a format suitable for visualization tools without additional transformation.
* Planned the structure of the dashboards including **Quick Stats**, **Nation Overview**, **Compare Powers**, and **Coalition Builder**.
* Created layout sketches and storyboard plans to guide the development of interactive dashboards.

# Deliverables

* **generate_kpis.ipynb** – script used to generate KPIs
* **military_final.xlsx** – final dataset containing engineered metrics
* **dashboard_storyboard.md** – dashboard layout and design plan


---

# Output

The primary output of this milestone is an **analysis-ready dataset enriched with KPIs** that can be directly used for dashboard development.

The dashboard design blueprint created in this stage ensures that the full dashboard suite can be implemented efficiently in the next milestone.
